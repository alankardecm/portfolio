# 🚀 Portfolio — Alan Moreira

Site portfólio profissional de **Alan Moreira** — Consultor BI, IA & Dados | AM Consultoria.

## 🤖 Desenvolvido com IA

Este site foi **criado inteiramente com o Gemini (Antigravity)** do Google DeepMind, via pair programming assistido por IA.

- **IA utilizada**: Gemini / Antigravity (Google DeepMind)
- **Método**: Pair programming conversacional — o código foi gerado, refinado e deployado em sessões interativas
- **Data de criação**: Fevereiro de 2026
- **Deploy**: Vercel → [alan-moreira-portfolio.vercel.app](https://alan-moreira-portfolio.vercel.app/)

## 🛠️ Stack Técnica

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica com SEO completo (Open Graph, JSON-LD, Twitter Cards) |
| CSS3 (Vanilla) | Design system com variáveis CSS, glassmorphism, animações, responsivo |
| JavaScript (Vanilla) | Interatividade, particle system, matrix rain, typed text, scroll reveals |
| Vercel | Hosting e deploy |

## ✨ Features do Site

- 🎨 Design premium dark theme com efeitos visuais (glitch, scanlines, HUD corners)
- 🖥️ Terminal fake animado com typing effect
- 📊 Seção de skills com barras de progresso animadas
- 🗂️ Portfólio de projetos com filtros por categoria (BI, IA, Dev, Data)
- 📈 Seção de impacto com contadores animados
- 📱 Totalmente responsivo (mobile-first)
- 🔍 SEO otimizado (Schema.org, meta tags, canonical URL)
- ⚡ Performance otimizada (sem frameworks pesados)
